6. 文件类型说明

Linux 一切皆文件，主要类型包括：

  • -：普通文件
  • d：目录文件
  • l：符号链接（相当于快捷方式）
  • c：字符设备文件
  • b：块设备文件
  • s：套接字文件
  • p：管道文件
